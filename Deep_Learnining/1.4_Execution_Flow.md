
## Deep Learning – Basic Working (Intuition)

### Step 1: Input

It starts with simple inputs that we provide.

* Example: x₁, x₂
* At this stage, these are just raw values

---

### Step 2: First Combination

The model tries a combination of the inputs.

* It mixes x₁ and x₂ in some way
* This produces a new value

This new value is:

> a new representation of the original data

---

### Step 3: Build Representations Layer by Layer

* The output of the first combination becomes input for the next
* The model again creates a new combination

So the process becomes:

x₁, x₂ → first representation → second representation → ...

Each step:

* transforms the data
* builds a slightly more meaningful representation

---

### Step 4: Final Output

After multiple such transformations:

* The model produces a final output
* This is its prediction

---

### Step 5: Compare with Actual Output

* The predicted output is compared with the actual (true) output
* The difference between them is the error

---

### Step 6: Self-Correction

Based on the error:

* The model adjusts how it makes combinations
* It improves step by step

Over time:

* better representations are learned
* predictions become more accurate

---

## Core Flow

Input → Combination → Representation →
Repeated Transformations → Final Output →
Compare with Actual → Correct Itself

---

## Key Insight

The model does not directly jump from input to output.

It:

* builds intermediate representations
* refines them step by step
* learns from mistakes

---

## One Line Summary

Deep learning works by repeatedly transforming input into better representations and correcting itself using error until it predicts the right output.
