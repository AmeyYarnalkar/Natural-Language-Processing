# Loss Functions for Regression

## 1. Mean Squared Error (MSE)

| Key              | Value                                                              |
| ---------------- | ------------------------------------------------------------------ |
| Definition       | Average of squared differences between predicted and actual values |
| Formula (Single) | (y − ŷ)²                                                           |
| Formula (Batch)  | (1/n) Σ (yᵢ − ŷᵢ)²                                                 |
| Behavior         | Penalizes large errors heavily                                     |
| Advantages       | Smooth and easy to optimize; Strong penalty for large errors       |
| Disadvantages    | Sensitive to outliers; Can be dominated by large errors            |

---

## 2. Mean Absolute Error (MAE)

| Key              | Value                                  |
| ---------------- | -------------------------------------- |
| Definition       | Average of absolute differences        |
| Formula (Single) | |y − ŷ|                                |
| Formula (Batch)  | (1/n) Σ |yᵢ − ŷᵢ|                      |
| Behavior         | Treats all errors equally              |
| Advantages       | Robust to outliers; Easy to interpret  |
| Disadvantages    | Not smooth at zero; Slower convergence |

---

## 3. Root Mean Squared Error (RMSE)

| Key              | Value                                               |
| ---------------- | --------------------------------------------------- |
| Definition       | Square root of MSE                                  |
| Formula (Single) | √((y − ŷ)²)                                         |
| Formula (Batch)  | √((1/n) Σ (yᵢ − ŷᵢ)²)                               |
| Behavior         | Same as MSE but scaled back to original units       |
| Advantages       | Same unit as output; Penalizes large errors         |
| Disadvantages    | Still sensitive to outliers; Adds extra computation |

---

## 4. Huber Loss

| Key              | Value                                                                   |
| ---------------- | ----------------------------------------------------------------------- |
| Definition       | Combines MSE and MAE                                                    |
| Formula (Single) | if |y − ŷ| ≤ δ → (1/2)(y − ŷ)² <br> if |y − ŷ| > δ → δ(|y − ŷ| − 1/2 δ) |
| Formula (Batch)  | (1/n) Σ Huber(yᵢ, ŷᵢ)                                                   |
| Behavior         | Smooth near zero, robust for large errors                               |
| Advantages       | Less sensitive to outliers than MSE; Smooth optimization                |
| Disadvantages    | Requires tuning parameter (δ); Slightly more complex                    |

---

# Intuition (Very Important)

* **MSE** → cares a lot about big mistakes
* **MAE** → treats all mistakes equally
* **RMSE** → same as MSE but easier to interpret
* **Huber** → balance between MSE and MAE

---

# When to Use What

### Use MSE when:

* You want to heavily penalize large errors
* Data is relatively clean

---

### Use MAE when:

* Data has **outliers**
* You want robustness

---

### Use RMSE when:

* You want error in **same units as output**

---

### Use Huber when:

* You want a **balance**
* You expect some outliers but not too many

---

# Final Practical Rule

> Start with MSE → if outliers cause issues → move to Huber or MAE

---

# One Line Summary

Different regression losses mainly differ in how they treat large errors—MSE amplifies them, MAE ignores scale, and Huber balances both.
