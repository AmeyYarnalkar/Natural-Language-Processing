
## Neural Network – Terminology (Based on the Execution Flow)

### 1. Model / System

The complete setup is called a **Neural Network** (also referred to as a Perceptron model in its simplest form).

It takes input, transforms it through multiple stages, and produces an output.

---

### 2. Input Layer

* This is where we provide the initial data
* Example: x₁, x₂

It does not perform any transformation.
It only passes the input forward.

---

### 3. Hidden Layers

* These are the intermediate stages between input and output
* Here, the actual transformations happen

Each hidden layer:

* takes input
* creates a new representation
* passes it forward

So the flow becomes:

Input → Hidden Layer 1 → Hidden Layer 2 → ...

---

### 4. Output Layer

* This is the final stage
* It produces the final prediction

Example:

* a number (regression)
* a class label (classification)

---

## Activation Function


* The **combination step** creates a value
* The **activation function** modifies that value

So:

> Activation function does not create the representation
> It **shapes or transforms** the representation

Why it matters:

* Without it, everything behaves like a simple linear system
* With it, the model can learn complex patterns

---

## 5. Weights (Very Important Term)

* These control how inputs are combined
* They decide the importance of each input

You can think:

> weights = how much attention each input gets

---

## 6. Bias

* An additional value added during combination
* Helps shift the result

It allows the model to:

* adjust more flexibly
* not depend strictly on input values

---

## 7. Prediction

* The output produced by the model

---

## 8. Loss (Error)

* The difference between predicted output and actual output

This tells:

> how wrong the model is

---

## 9. Learning / Updating

* The model adjusts weights and bias based on the error
* This is how it improves over time

---

## Full Flow with Terminology

Input Layer →
Hidden Layers (with weights + bias + activation function) →
Output Layer →
Prediction →
Compare with Actual →
Loss →
Update weights and bias

---

## Final Clean Understanding

* Input layer → receives data
* Hidden layers → build representations
* Activation function → transforms values to allow complexity
* Weights & bias → control how data is combined
* Output layer → produces final result
* Loss → measures error
* Learning → improves the model

---
