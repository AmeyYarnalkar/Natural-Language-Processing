#  N-Grams in NLP

## **Core Idea**

Instead of treating words independently (like BOW), we consider **sequences of words** to capture limited context.

---

## **Problem (with BOW)**

* Loses word order
* Fails on phrases like:

  * “not good” vs “good”
* No context understanding

---

## **How it Works**

An **n-gram** is a sequence of *n consecutive words*.

* **Unigram (n=1)** → single words → same as BOW
* **Bigram (n=2)** → pairs of words
* **Trigram (n=3)** → triplets

---

###  Example

Sentence:

```
I love machine learning
```

* Unigrams:

  ```
  [I, love, machine, learning]
  ```

* Bigrams:

  ```
  [I love, love machine, machine learning]
  ```

* Trigrams:

  ```
  [I love machine, love machine learning]
  ```

---

##  How N-Grams Improve Things

### 1) Capture Local Context

* “not good” becomes a **feature**
* Now model distinguishes:

  * “good” ✅
  * “not good” ❌

👉 This is the *biggest improvement*

---

### 2) Preserve Word Order (partially)

* BOW ignores order completely
* N-grams keep **neighbor relationships**

---

### 3) Better Representation for Phrases

* “New York”
* “machine learning”

👉 These become meaningful features

---

##  Limitations (very important)

This is where most people misunderstand.

### 1) Still No True Semantics

* “great” vs “awesome” → unrelated
* “buy now” vs “purchase immediately” → different

👉 No understanding of meaning, only patterns

---

### 2) Vocabulary Explosion

* Bigrams/trigrams grow **very fast**

Example:

* 10k words →

  * unigrams: 10k
  * bigrams: ~100M possibilities

👉 leads to:

* huge feature space
* more sparsity

---

### 3) Data Dependency

* Rare phrases may never appear
* Model can’t learn them

---

### 4) Still Shallow Context

* Only looks at nearby words
* Misses long-range meaning

Example:

```
I thought the movie would be good, but it was terrible
```

👉 bigrams won’t fully capture this contrast

---

##  Key Insight 

* **BOW** → no context
* **N-grams** → *local* context
* **Embeddings/Transformers** → *real semantic understanding*

👉 N-grams are a **patch**, not a full solution
