# NLP Notes: Lemmatization

## 1. Core Idea

**Lemmatization** is the process of converting a word to its **dictionary base form (lemma)** using linguistic knowledge such as vocabulary and grammar.

---

## 2. Problem

Words appear in multiple grammatical forms:

* run, running, ran
* study, studies, studying
* good, better

Challenges:

* increases vocabulary size
* creates inconsistency in representation
* stemming may produce incorrect or invalid words

There is a need for a method that:

* preserves meaning
* returns valid base forms

---

## 3. How it Works

```text id="l2p9qd"
Input Text
 → Tokenization
 → POS Tagging (optional but important)
 → Dictionary lookup (WordNet or similar)
 → Output lemmas
```

Process:

1. Tokenize the text
2. (Optionally) assign POS tags
3. Use a lexical database to find base form
4. Return valid root words

---

## 4. Examples

| Word           | Lemma |
| -------------- | ----- |
| running (verb) | run   |
| studies        | study |
| better         | good  |
| mice           | mouse |

Without POS:

* running → running (incorrect)

With POS:

* running (verb) → run (correct)

---

## 5. Code Snippet

Using NLTK:

```python id="q8t6nv"
from nltk.stem import WordNetLemmatizer
from nltk.tokenize import word_tokenize

# nltk.download('wordnet')
# nltk.download('punkt')

text = "running studies better"

lemmatizer = WordNetLemmatizer()
tokens = word_tokenize(text)

lemmas = [lemmatizer.lemmatize(word) for word in tokens]

print(lemmas)
```

---

## 6. Variations

* **WordNet Lemmatizer (NLTK)**
  Uses lexical database

* **spaCy Lemmatizer**
  Faster, includes built-in POS tagging

* **Rule-based Lemmatizers**
  Simpler, less accurate

---

## 7. Limitations

* Slower than stemming
* Requires POS tagging for best results
* Dependent on dictionary coverage
* May not handle complex context perfectly

---

## 8. Comparison

Lemmatization vs Stemming:

* Lemmatization:

  * Uses vocabulary + grammar
  * Produces valid words
  * More accurate
  * Slower

* Stemming:

  * Rule-based
  * Produces truncated forms
  * Faster
  * Less accurate

---

## 9. When to Use / When Not to Use

Use when:

* meaning accuracy is important
* working with NLP tasks requiring interpretability
* dealing with semantic analysis

Avoid when:

* speed is critical
* approximate normalization is sufficient

---

## 12. Key Insight

Lemmatization focuses on linguistic correctness rather than simple reduction. It ensures that different word forms are mapped to meaningful base words, improving both interpretability and model performance.
